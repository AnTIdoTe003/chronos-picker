# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Chronos Picker
- Full IANA timezone support with default to Asia/Kolkata (IST)
- Modern, accessible date and time picker component
- Calendar view with month/year navigation
- Time picker with 12-hour and 24-hour formats
- Optional timezone selector dropdown
- Date range restrictions (min/max dates)
- Comprehensive keyboard navigation support
- WCAG 2.1 AA accessibility compliance
- Responsive design for mobile and desktop
- Dark mode support
- TypeScript definitions
- Complete documentation with examples
- Demo application showcasing all features
- Support for multiple date/time formats using Luxon
- Smooth animations and transitions
- Click-outside and escape key to close
- Proper focus management
- ARIA labels and attributes
- Custom CSS class support
- Disabled state support

### Documentation
- Complete README with API reference
- Getting Started guide
- Usage examples for common scenarios
- Integration guides for React Hook Form and Formik
- Contributing guidelines
- License (MIT)

### Developer Experience
- TypeScript support with full type definitions
- ESLint configuration
- Vite for fast development and building
- Example demo application
- Proper package exports (ESM and UMD)
