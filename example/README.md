# Chronos Picker Demo

This is a demo application showcasing all features of the Chronos Picker component.

## Running the Demo

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:5173`

## Examples Included

The demo showcases:

1. **Basic Usage** - Simple date and time picker with default timezone
2. **Date Only** - Pick only the date without time
3. **Timezone Selector** - Select timezone along with date and time
4. **Date Range Restrictions** - Limit selectable dates
5. **12-Hour Format** - Time picker with AM/PM
6. **Multiple Timezones** - View selected time across different timezones

## Building the Demo

```bash
npm run build
```

The built files will be in the `dist` directory.
